dotfiles
====================
'dotfiles' includes .vimrc, .gvimrc, .bashrc and so on.

#### Prerequirements
+ OS: MacOSX or Linux(Ubuntu, CentOS...)
+ git
+ (C++) clang-format 3.4 or later

(optional)
+ ag: The Silver searcher (for Unite grep)

### Usage

```bash
$ cd

$ git clone https://github.com/rkaneko/dotfiles.git

$ cd dotfiles

$ ./setup.sh

$ vi

on vim
:NeoBundleInstall
:q!
```
