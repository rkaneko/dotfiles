dotfiles
====================
This is my dotfiles .

Usage
-----

### on console ###
    $ cd

    $ git clone https://github.com/rkaneko/dotfiles.git

    $ cd dotfiles

    ## for Shougo/neobundle.vim
    $ mkdir -p ~/dotfiles/.vim/bundle
    $ git clone https://github.com/Shougo/neobundle.vim ~/dotfiles/.vim/bundle/neobundle.vim

    $ ./setup.sh

    $ vi

    on vim
    :NeoBundleInstall
    :q!


End
